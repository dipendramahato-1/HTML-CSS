# Job Application Form (Web)

This is a responsive Job Application Form built using HTML and CSS.

The project focuses on creating a clean and structured form UI with proper alignment and user-friendly design.

---

## 🚀 Features
- Responsive form layout using CSS Grid  
- Input fields for user details  
- Dropdown selection for job roles  
- File upload option (CV upload)  
- Textarea for address input  
- Styled submit button with hover effects  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  

---

## 📚 What I Learned
- Creating structured forms using HTML  
- Using CSS Grid for layout design  
- Styling input fields, buttons, and forms  
- Handling responsiveness using media queries  
- Improving UI with spacing and alignment
